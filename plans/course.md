# Course Outline

An end-to-end course on building traceable, branded, multimedia technical proposals with agentic tools.

> **Legend:** 🔹 **Adv** marks advanced modules.

---

## 1. Overview: What you will learn

The end-to-end workflow and toolchain:

- **PraxiumAI**
- **Cowork / Codex Desktop**
- **Claude Design**
- **Claude Code / Codex**
- **Replit**

## 2. Co-create basic proposals

*Tool: Cowork / Codex Desktop*

- Prompt-driven agentic ingest and parse of real datasheets.
- Co-create a basic proposal with agents, with light branding (Claude Design).
- Hit the review & verifiability wall — hallucination and untraceable claims.

## 3. (🔹 Adv) Rigorous proposals

*Tools: Claude Code & Codex*

- Understand agentic data formats and work with PraxiumAI agentic data.
- Apply the **GLCCVP** framework: Gather → Learn → Compare → Co-Create → Verify → Propose via coding agents.
- Outcome: traceable, branded, multimedia proposals.

## 4. Branding deep-dive

*Tool: Claude Design*

- Professional, on-brand docs and web pages: color, typography, and logo.
- Brand tokens.
- A reusable style / Skill.

## 5. (🔹 Adv) Publish interactive web proposals

- Turn technical content into live, interactive webpages using multiple agents and sub-agents: **Replit + Claude Code + Codex**.
- When to use Replit Agents vs. Claude Code / OpenAI Codex.

## 6. Rich-media assets

*Tools: Replit / Cowork / Codex*

- Technical marketing assets beyond docs.
- De-AI the output: kill em-dashes, repetitive rhythm/cadence, hedging, clichés, and emoji overuse.
- Storyboarding.

## 7. Personalized learning using PraxiumAI

- Tailor learning to a specific customer/persona, use case, and goal.
- Understand learning-science-driven content creation.
- Downloading and using PraxiumAI data for your agentic workflow
- Verifiability & traceability.

## 8. Technical diagrams

- Generate diagrams using Cowork / Codex Desktop.
- Generate accurate diagrams with verifiability using PraxiumAI.
- Overcome agent limitations:  wrong labels, broken layout.

## 9. Quality control

*Tool: Skills*

- Add QC guardrails for verifiability, traceability, and accuracy.
- Understand the limits of Skills and when to use prompts instead

## 10. (🔹 Adv) Build your own agentic workflow

- Design an end-to-end pipeline on PraxiumAI data for your own docs.
- The way we built PraxiumAI.

## 11. Tool & cost tradeoffs

- How to decide between Chat / CoWork / Coding agents.
- Token/cost vs. quality vs. speed.
- When a simpler tool wins.

## 12. Iterate from feedback

- Version proposals using GitHub as customer requirements change.
- Capture review comments and re-run the pipeline without starting over.
