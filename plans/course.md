# Course Outline

An end-to-end course on building traceable, branded, multimedia technical proposals with agentic tools.

> **Legend:** **Adv** marks advanced modules.

---

## 1. Overview: What you will learn

The end-to-end workflow and toolchain:

- **PraxiumAI**
- **Cowork / Codex Desktop**
- **Claude Design**
- **Claude Code / Codex**
- **Replit**

## 2. Co-create basic proposals

*Tool: Cowork / Codex Desktop*

- Use prompt-driven agents to ingest and parse real datasheets.
- Co-create a basic proposal with agents, with light branding applied through Claude Design.
- Hit the review and verifiability wall, where hallucination and untraceable claims surface.

## 3. (Adv) Rigorous proposals

*Tools: Claude Code & Codex*

- Understand agentic data formats and work directly with PraxiumAI agentic data.
- Apply the **GLCCVP** framework: Gather → Learn → Compare → Co-Create → Verify → Propose, all driven by coding agents.
- Produce the outcome: traceable, branded, multimedia proposals.

## 4. Branding deep-dive

*Tool: Claude Design*

- Create passable, on-brand docs and web pages with consistent color, typography, and logo.
- Define and reuse brand tokens.
- Package it all into a reusable style or Skill.

## 5. (Adv) Publish interactive web proposals

- Turn technical content into live, interactive web pages using multiple agents and sub-agents: **Replit + Claude Code + Codex**.
- Learn how to decide between Chat, Desktop agents (Cowork / Codex), and Coding agents.

## 6. Rich-media assets

*Tools: Replit / Cowork / Codex*

- Stand out by being different.
- Create technical marketing assets that go beyond docs.
- De-AI the output by killing em-dashes, repetitive rhythm and cadence, hedging, clichés, and emoji overuse.
- Build out storyboarding.

## 7. Personalized learning using PraxiumAI

- Tailor learning to a specific customer or persona, use case, and goal.
- Understand learning-science-driven content creation.
- Download and use PraxiumAI data for your agentic workflow.
- Ensure verifiability and traceability throughout.

## 8. Technical diagrams

- Generate diagrams using Cowork / Codex Desktop.
- Generate accurate diagrams with verifiability using PraxiumAI.
- Overcome agent limitations such as wrong labels and broken layout.

## 9. Quality control

*Tool: Claude Skills*

- Practice spec-driven development.
- Build evals for technical definitions, facts, procedures, troubleshooting, math, quants, and formulas, as well as visuals and coherence.
- Add quality-control guardrails for verifiability, traceability, and accuracy.
- Understand the limits of Skills and when to use prompts instead.

## 10. (Adv) Build your own agentic workflow

- Design an end-to-end pipeline on PraxiumAI data for your own docs.
- Extend and build on PraxiumAI's agentic system.

## 11. (Bonus) Tool & cost tradeoffs

- Weigh token and cost against quality and speed.
- Recognize when a simpler tool wins.

## 12. (Bonus) Iterate from feedback

- Version proposals using GitHub as customer requirements change.
- Capture review comments and re-run the pipeline without starting over.
